🎯 PROBLEMAS CONOCIDOS:

• python3 -m venv no funciona en Codespace
  └─ SOLUCIÓN: Usa bash setup.sh (instala directamente)

• npm install toma 1-2 minutos
  └─ SOLUCIÓN: Normal, espera a que termine

• CORS error cuando conectas frontend
  └─ SOLUCIÓN: Asegúrate que backend está corriendo
