┌──────────────────────────────────────────────────────────────────────┐
│  PASO 7: PRUEBAS MANUALES EN LA UI                                  │
└──────────────────────────────────────────────────────────────────────┘

    □ Crear Perfil
      1. Click en "Configurar Perfil"
      2. Llenar formulario
      3. Click "Guardar"
      ✓ Deberías estar en el dashboard

    □ Crear Producto
      1. Click en "Productos" en menú lateral
      2. Click botón "+" o "Nuevo Producto"
      3. Llenar formulario:
         - Nombre: iPhone 15 Pro
         - Código: IP15PRO001
         - Precio: 999.99
         - Costo: 750.00
         - Stock: 10
         - Categoría: Smartphones
      4. Click "Guardar"
      ✓ Producto aparece en la lista

    □ Crear Venta
      1. Click en "Nueva Venta" (botón verde)
      2. Agregar productos a la venta
      3. Llenar datos del cliente
      4. Seleccionar método de pago
      5. Click "Completar Venta"
      ✓ Venta creada, stock actualizado

    □ Ver Estadísticas
      1. Click en "Dashboard"
      2. Verificar:
         - Total de ventas del día
         - Productos en inventario
         - Gráfico de ventas
      ✓ Estadísticas correctas


┌──────────────────────────────────────────────────────────────────────┐
│  VERIFICAR CONSOLA DEL NAVEGADOR                                    │
└──────────────────────────────────────────────────────────────────────┘

    1. Presionar F12 en el navegador
    2. Ir a pestaña "Console"
    3. Verificar:
       ✓ No hay errores rojos
       ✓ Solo mensajes informativos azules (si hay)

    Si ves errores:
       ✗ TypeError, ReferenceError → Hay un bug
       ✗ CORS errors → Backend no está corriendo
       ✗ Network errors → Backend no responde


┌──────────────────────────────────────────────────────────────────────┐
│  VERIFICAR DOCUMENTACIÓN DE LA API                                  │
└──────────────────────────────────────────────────────────────────────┘

    1. Abrir: http://localhost:8000/docs
    2. Deberías ver interfaz Swagger UI
    3. Probar endpoints:
       - GET /health
       - POST /api/profiles
       - POST /api/auth/login
       - GET /api/products

┌─ "puerto 8000 ya está en uso"
│  $ lsof -i :8000       # Ver qué lo usa
│  $ kill -9 <PID>       # Matar proceso
└─

┌─ "comando no encontrado: uvicorn"
│  $ cd backend
│  $ source venv/bin/activate
│  $ pip install -r requirements.txt
└─

┌─ "npm: command not found"
│  $ Instalar Node.js desde https://nodejs.org
└─

┌─ "python3: command not found"  
│  $ Instalar Python desde https://python.org
└─

┌─ Frontend no carga
│  $ rm -rf node_modules package-lock.json
│  $ npm install
│  $ npm run dev
└─

┌─ Base de datos bloqueada
│  $ cd backend
│  $ rm inventory.db
│  $ python3 init_db.py
└─
