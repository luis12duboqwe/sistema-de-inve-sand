# Política de Seguridad del Proyecto

Este repositorio sigue una política de divulgación responsable.

## Versiones soportadas

| Versión | Soporte de seguridad |
| --- | --- |
| 2.0.x | ✅ Sí |
| 1.x | ❌ No |

## Cómo reportar vulnerabilidades

No abras issues públicos para vulnerabilidades.

Reporta por correo a: seguridad@softmobile.com

Incluye, si es posible:

- Tipo de vulnerabilidad.
- Archivos/rutas afectadas.
- Pasos de reproducción.
- Impacto esperado.
- Prueba de concepto (opcional).

## Tiempos de respuesta

- Confirmación inicial: hasta 48 horas.
- Priorización y plan de mitigación: según severidad.

## Alcance

Esta política aplica a este repositorio y sus despliegues oficiales.
